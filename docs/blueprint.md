# **App Name**: Wont Stop Moving

## Core Features:

- Public Website & Inquiries: Provide a public-facing website with information about services, company values, and contact forms for general inquiries.
- Quote Request & Job Application System: Enable unauthenticated users to request detailed moving quotes and submit job applications directly into Firestore, with resume upload to Cloud Storage.
- Secure User Authentication & Role-Based Access: Implement Firebase Authentication for customer, employee, and admin roles, ensuring secure, role-specific access to dashboard functionalities.
- Customer Dashboard: Authenticated customers can track the status of their active moves, view their order history, and submit new quote requests.
- Employee Job Management: Allow authenticated employees to view their assigned jobs, update job statuses in real-time, and manage their personal availability and profiles.
- Admin Control Panel: Provide administrators with comprehensive tools to manage jobs (assignment, status override), employees (profiles, status), customer inquiries, and job applications.
- AI-Enhanced Job Note Tool: Offer employees an AI-powered tool within job details to suggest structured notes or summarize key observations, streamlining communication and record-keeping.

## Style Guidelines:

- Background color: A deep charcoal gray (`#171B1F`) provides a modern and professional canvas for content, subtly referencing the deep navy.
- Primary accent color: Deep navy (`#0A1628`), representing reliability and professionalism, used for key structural elements and headings, aligning with the brand identity.
- Secondary accent color: Vibrant electric orange (`#FF6B2B`), serving as a dynamic contrast for interactive elements, calls to action, and highlighting important information.
- Semantic colors: Green (`#10B981`) for success, amber (`#F59E0B`) for warnings, and red (`#EF4444`) for error states, providing clear and consistent user feedback.
- Headlines: 'Syne' (sans-serif) for bold, kinetic typography, giving a strong and modern voice to headings. Body text: 'DM Sans' (sans-serif) for legibility and a contemporary feel.
- Utilize a consistent set of clean, modern, outline-style icons to visually reinforce actions, services, and navigation cues across the application.
- Adaptive, mobile-first design ensuring full responsiveness across all device sizes. Admin and employee dashboards feature a dark sidebar navigation, while public and customer pages maintain a lighter, airy aesthetic.
- Content presented within rounded cards with subtle shadows, providing a contemporary, organized, and slightly tactile user interface.
- Homepage hero features dynamic, kinetic typography to captivate visitors. Subtle and smooth transitions are used for page loads and component state changes for a refined user experience.
